# Work Day Planner

## Description 
Work planner created for busy schedules. Add your todo to the corresponding time slot, and click the save button on the right side. Tasks then will be saved in local storage information will remain in the time block even if refreshed!

Text blocks will show a color of grey, green, or red depending on if the task is in the past, present, or future. 

Grey stands for the past.
Red stands for the present.
Green stands for the future.
